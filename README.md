# GD2_Resonnance_IlanaLinares

Ce début de prototype se nomme Resonnance et explore en partie le minimalisme. Le jeu se concentre sur les feedback avec deux jeux en vois d'amélioration.
Le premier jeu est un jeu Game and Watch le but est simple : Récupérer un maximum de cube afin de faire le score le plus élever avec des collectibles qui vont de plus en plus vite ainsi que des Fast Enemy qui servent de bonus. Le joueur posséde trois vie. Le jeu se concentre sur les feedback et l'esprit de compétition.
Le deuxieme est jeu est un jeu arcade ou des boules de différentes couleurs spawns et il faut les envoyer sur d'autres boules qui possédent bien sur la bonne couleur. comme contrainte : Un nombre de cout limité avec possibilité de tomber sur des boules bonus qui rajouteront 5 couts supplémentaires. Les rebonds sur les coté sont possible afin d'augmenter la stratégie du joueur.
De l'habillage léger a étais mis en place et du sound design également.
